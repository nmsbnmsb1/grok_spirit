## ‚ú® Features (No Built-In Editor)

- üîé **Prompt Capture** ‚Äì Automatically records the JSON Grok uses whenever a video finishes
- üìã **Clipboard Injection** ‚Äì Reads plain text/JSON from your clipboard and auto-fills Grok‚Äôs textarea
- üóÇÔ∏è **Folder & Sequence Naming** ‚Äì Define the exact folder + 3-digit sequence for downloads (e.g., `Grok/demo/007`)

## ü™∂ Edition Overview 

This No-Editor Edition is tailored for two reliable workflows:

1. **Inspect prompts** ‚Äì When Grok completes a video, the panel shows the captured JSON (read-only) plus status/time metadata.
2. **Replay prompts** ‚Äì Edit JSON externally, copy it, then press **Clipboard**; Grok‚Äôs textarea updates instantly, ready to regenerate.

The panel also exposes two text inputs (folder + sequence) so downloads land exactly where you expect‚Äîe.g., `Grok/my-project/007.mp4` and the matching `.json`.

## üìñ Usage Guide

### Minimal Workflow (Clipboard Driven)

1. Generate a video on `grok.com/imagine`.
2. Once the request finishes, the panel shows the captured JSON plus a status chip (processing/failed/completed).
3. To change the prompt, copy it into your favorite editor, tweak it, then copy the new JSON back to your clipboard.
4. Click **Clipboard** in the panel; the textarea on Grok updates immediately so you can press Generate.
5. Fill in `Folder` and `Sequence`, then click **Download** to save `Grok/<folder>/<sequence>.{mp4,json}` (HD requested automatically when possible).

> üí° All prompt editing happens outside Grok in this build; the panel only handles display, injection, and downloads.

### Clipboard Tips

- Accepts plain text, structured JSON, or multi-locale JSON (the `en` branch is injected).
- After injection, Grok treats the text as if you pasted it manually, so you can instantly generate again.

## üé¨ Video Metadata Processing

A companion Python tool helps batch-process downloaded MP4/meta files:

### What it does
- **Metadata embedding** ‚Äì Uses FFmpeg to embed JSON metadata into MP4 files
- **Smart renaming** ‚Äì Groups videos by prompt and assigns priority/version labels
- **Batch processing** ‚Äì Handles entire directories at once

### Quick setup
1. **Prerequisites**: Python 3.10+, FFmpeg
2. **Install deps**:
   ```bash
   cd grok_video_processor
   pip install -r requirements.txt
   ```
3. **Run**:
   ```bash
   python meta_video.py
   ```

### File organization
- Groups similar prompts
- Creates priority numbers (P1, P2, ‚Ä¶) and version numbers (v1, v2, ‚Ä¶)
- Final naming example: `grok_video_[uuid]_P1_v1.mp4`

**üìñ Full documentation**: see [`grok_video_processor/README.md`](grok_video_processor/README.md).
